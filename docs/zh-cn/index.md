# 关于 LPSub Fonts

[Google Fonts](https://developers.google.com/fonts/docs/getting_started) 有 Noto 系列，还有 Ubuntu 等一堆 webfont 可以调用，但是 Notorious Sans 系列在浏览器里面渲染效果没有更纱好，所以就有了这个项目。
